# Scrivener-Cobalt-Theme
A theme for Srivener based on the Cobalt2 theme from Wes Bos

# Install
1. Download the latest release
2. Open Scrivener
3. Go to File -> Options
4. In the bottom-left corner of the window, click `Manage`
5. Click `Load Theme from file`
6. Select the downloaded file and click the `Open` button
7. Click `Ok`
8. Restart Scrivener
